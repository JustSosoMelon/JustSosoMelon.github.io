1环境管理

1.1create env

  1. conda create --name env_name

启动
Linux, OS X: source activate env_name
Windows: activate env_name

关闭
Linux, OS X: source deactivate
Windows: deactivate

1.2 list envs

  1. conda info --envs
  2. conda env list
  3. conda info -e

1.3Remove an environment

  1. conda remove --name env_name --all

1.4Share an environment

  1. conda env export > environment.yml
  2. # NOTE: 包含env所需的包.
  3. # 创建另外一个包含这些包的env:
  4. conda env create -f environment.yml

1.5创建一个相同的环境

The command conda list --explicit produces a spec list
The command conda list --explicit > spec-file.txt creates a file containing this spec list in the current working directory.

conda create --name MyEnvironment --file spec-file.txtuses the spec file to create an identical environment on the same machine or another machine.
The command conda install --name MyEnvironment --file spec-file.txtadds these packages to an existing environment.
